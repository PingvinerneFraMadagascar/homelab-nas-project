# Secure Homelab NAS Project

This repository documents the design, deployment, and operation of a secure
homelab NAS environment built for learning, experimentation, and limited
personal services.

## Objectives
- Learn practical system administration and cybersecurity concepts
- Deploy containerized services (Docker)
- Experiment with virtualization in a safe environment
- Implement secure storage and access controls
- Document architectural decisions and trade-offs
